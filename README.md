# Numerix Pro — Zero-Config Deploy (Demo Mode)

This build deploys **without any env vars**. Payments are hidden unless you set `NEXT_PUBLIC_ENABLE_PAY=1`.
Numerology & Vastu engine works out of the box.

## Quick start
```bash
npm install
npm run dev
```
Optional:
- Enable payments later by setting env vars and `NEXT_PUBLIC_ENABLE_PAY=1`.
